Methodology & Data Sources

Calculations use census and ACS housing data for income and home values. Target borrower income is set to the highest AMI level where an affordability gap exists for each state. Affordability is modeled using a 35% front-end DTI ratio, 3% down payment, 7% fixed interest rate, and estimated state-level averages for property taxes, homeowners insurance, and PMI.

“Affordability Gap” represents the difference between the state median home value and the maximum affordable home price for a household at the target borrower AMI — the gap that Homium’s shared appreciation mortgage program directly closes.

Disclosure: These figures are illustrative estimates based on standardized assumptions. Actual affordability will vary based on individual borrower income, credit profile, interest rates, loan terms, taxes, insurance costs, and local market conditions. This analysis is provided for informational purposes only and does not constitute a credit decision, underwriting determination, or offer to lend.
